rearrange me
&nbsp; &nbsp; &nbsp; &nbsp; tenderly
one petal here    another 
there 	
like Spring
&nbsp; &nbsp; &nbsp; fingers		
&nbsp; &nbsp; &nbsp; lose the grip
but don’t &nbsp; let go

the wind and rain
never felt so good
&nbsp; &nbsp; and desire	
the whole &nbsp; of my heart
&nbsp; between &nbsp; sugared brick
and a salted sky &nbsp; swimming
&nbsp; &nbsp; &nbsp; in gold

&nbsp; &nbsp; &nbsp; we find ourselves
the city lights &nbsp; floating &nbsp; &nbsp; like stars
in the East River
&nbsp; lapping &nbsp; in and out &nbsp; of view
&nbsp; yes &nbsp; &nbsp; that’s our view:
evenly spaced &nbsp; &nbsp; circles
&nbsp; &nbsp; cut with finite precision:
&nbsp; prisms &nbsp; &nbsp; of infinite possibility
&nbsp; &nbsp; through which 
&nbsp; &nbsp; &nbsp; we imagine
all &nbsp; &nbsp; the lives       
&nbsp; &nbsp; &nbsp;we &nbsp; could live:

&nbsp; “I’ll go where you go”:
&nbsp; just keep counting
&nbsp; the decimals with me:
I want to believe
&nbsp; &nbsp; the function converges
&nbsp; &nbsp; in the limit of time:
&nbsp; &nbsp; &nbsp; where you go 
&nbsp; &nbsp; &nbsp; &nbsp; I’ll go
